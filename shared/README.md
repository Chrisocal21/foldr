# Shared Images

Drop images here to share with the AI assistant.

## How to use:
1. Save/paste images into this folder or subfolders
2. Reference them in chat (e.g., "look at the image in shared/screenshot.png")

## Subfolders:
- `/screenshots` - UI screenshots, mockups
- `/logos` - Logo designs, branding
- `/reference` - Reference images, inspiration
