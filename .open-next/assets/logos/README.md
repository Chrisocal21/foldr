# Logo Files

Drop your logo files here. They will be served at `/logos/filename.png`

## Expected files:
- `logo.png` - Full logo with "Foldr" text (for header)
- `icon.png` - Just the icon (for favicon/PWA)
- `favicon.ico` - Browser tab icon (32x32)
- `apple-touch-icon.png` - iOS home screen (180x180)
- `icon-192.png` - PWA icon Android (192x192)  
- `icon-512.png` - PWA splash/install (512x512)
